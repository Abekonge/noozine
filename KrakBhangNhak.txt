Krak / Bhang Nhak

Brændt rødbedefarvet. Stank. Hissende krumlende kvælning. Vi var tusind der brændte pas den dag. Alle vi genkendte stod der. Og mange der var fremmede. Tusind og måske flere pas boblede plastik. Vi skreg: "FUUUUUUUUCK DAAAAAANMAAAARK" og stampede og hujede og tudede og klappede.

Og så kom politiet og bankede os og internerede os. Og så slap de os fri. Og så skred vi. Vi stemte den aften for at indlede forhandlinger med det postsekvente Grønlandske selvstyre. Vi arbejdede om dagene på programmerne, sugede strøm fra solen og vi havde ikke brug for nogen, og vi var bror og søster med alle.

Uge efter uge kom vi nye steder hen. Sevastopol Beograd Arden. I hver flække smed folk deres sager fra hænderne og kom med. Vi samledes og splittedes og kyssedes. Ved grænserne stod vi indtil vagterne klippede hegnene og kom med os. Soldater forsvandt i vores mængde.

Vi fjernede skrald, samlede det og byggede det. Seedbombede bygninger og veje og gødede med kemikalier fra lort og pis vi selv lavede. Høstede hvor vi kom. Såede og harvede til de næste. Vores sensornet dækkede det hele og jordens biomasse blev firedoblet i de år. Vi arbejdede for biomasse og frodighed, for planetær komplementaritet.

Så lukkede de os ned. Internerede os igen. Skød os. Men så var de også selv ude om det. Og planeten brændte og alle havde det mega sygt dårligt. Men så gjorde vi det igen lidt senere og de sagde undskyld og at de nu var blevet klogere og ikke ville gøre det igen. Og det var ok og nu kom de med.

Og så var vi pludselig 12 milliarder der alle gik på jorden. Vi grinte af de år vi boede i huse, af de 0.005% af menneskets tidsalder. Vi kaldte os selv Populus og blev noget mere pelsede.
